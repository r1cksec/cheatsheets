### Source
https://github.com/kornelski/7z  

### Install
```
sudo apt install p7zip-full
```

### Create 7 zip archive
```
7z a <resultZip> <fileOrDir>
```

### Create password protected zip archive
```
7z a -p <file>.7z <file>
```

### Unzip on place
```
7z x <file>.7z 
```

### Readable formats
```
vmdk
vhdx
```
